# CLAUDE.md

## Commands

- **Run tests:** `python3 -m pytest tests/ -v`
- **Run a single test:** `python3 -m pytest tests/test_parser.py::TestParseCartHtml::test_parse_sample_cart -v`
- **Install deps:** `pip3 install -r requirements.txt`
- **Install dev deps:** `pip3 install -r requirements-dev.txt`

## Python

Use `python3` (not `python`) — `python` is not available in this environment.

## Code style

Do not write obvious comments — comments that just restate what the code already says.
